# skaills Logo Prompts

> For use with Google Gemini / image generation tools.
> All logos should be on a **white background**.
> Brand name: **skaills**
> Taglines: "Intelligent Skills" (EN) | "Competencias Inteligentes" (ES) | "Compétences Intelligentes" (FR)

---

## 1. Clean Wordmark — Highlight "AI"

```
Design a modern, clean wordmark logo for "skaills" on a white background. The letters "AI" in the middle of the word should be visually distinct — use a bold gradient from electric blue to violet for "AI" while keeping "Sk" and "lls" in a dark charcoal or navy sans-serif typeface. Below the wordmark, include the tagline "Intelligent Skills" in a smaller, lighter weight font. Minimal, professional, tech-forward aesthetic. No icons, no illustrations. Flat design, suitable for web and print.
```

The word "skaills" should be in a modern sans-serif font with "AI" highlighted in a vibrant blue-purple gradient. Below, add the tagline "Intelligent Skills" in a thin, elegant typeface. Clean, flat design. Corporate yet innovative feel. Suitable for a SaaS platform.

## Color Palette Suggestions

| Role | Color | Hex |
|------|-------|-----|
| Primary dark | Deep indigo / navy | `#1e1b4b` or `#312e81` |
| AI accent | Electric blue-violet | `#6366f1` or `#818cf8` |
| Secondary accent | Teal / cyan | `#06b6d4` or `#22d3ee` |
| Text | Charcoal | `#1f2937` |
| Tagline | Medium gray | `#6b7280` |
| Background | White | `#ffffff` |

## Usage Notes

- For **favicon**: Prompts #4 (monogram) or #9 (minimal) work best
- For **website header**: Prompts #1, #6, or #9 — horizontal, compact
- For **presentations/splash**: Prompt #10 — stacked with all taglines
- For **app icon**: Prompts #2, #5, or #7 — icon + text combo
